```
Žmogus viską žinantis -
Tuščias viduj.
Žmogus visko bijantis -
Grius pakeliuj.
```
<!-- more -->
```

Ko žmogus troškia,
Tai tik paslaptis.
Kai nauja mintis nubloškia,
Geresnė tampa dabartis.

Viską žinant, nieko nėra.
Juk žmogus - būtybė paprasta,
Kai naujienos prispaudžia,
Žmogus save tik baudžia,
Nes viskas negali būt atrasta.
```
